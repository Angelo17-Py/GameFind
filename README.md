# 🎮 GameFind - Sistema de Comparación de Precios

¡Bienvenidos al desarrollo de **GameFind**! Este proyecto utiliza una arquitectura moderna diseñada para ofrecer una experiencia de usuario premium y un rendimiento excepcional.

## 🚀 Guía de Inicio Rápido

Para comenzar a trabajar en el proyecto, sigue estos pasos:

### 1. Requisitos Previos

Asegúrate de tener instalado [Node.js](https://nodejs.org/) (versión 18 o superior).

### 2. Instalación de Dependencias

```bash
npm install
```

### 3. Servidor de Desarrollo

Para levantar el proyecto localmente:

```bash
npm run dev
```

---

## 🛠️ Stack Tecnológico

- **Framework:** [React 18](https://react.dev/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Lenguaje:** [TypeScript](https://www.typescriptlang.org/)
- **Estilos:** CSS Vanilla (Siguiendo un sistema de diseño moderno)

---

## 🎨 Guía de Estilo y Diseño

En GameFind, la **estética es una prioridad**. Queremos que el usuario se sienta en una plataforma de última generación.

### Estructura de Archivos:

- `src/components/`: Componentes reutilizables (Botones, Cards, Inputs).
- `src/pages/`: Vistas principales de la aplicación.
- `src/assets/`: Imágenes, iconos y recursos estáticos.
- `src/index.css`: Definición del sistema de diseño global (variables CSS).
